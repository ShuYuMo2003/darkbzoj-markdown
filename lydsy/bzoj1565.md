## Description: 




## Sample Input: 
```
3 2
10 0
20 0
-10 0
-5 1 0 0
100 1 2 1

```

## Sample Output: 
```
25
//在样例中, 植物P1,1可以攻击位置(0,0), P2, 0可以攻击位置(2,1)。
```
