# [Usaco2008 Jan]电话网络 - 题目 - Universal Online Judge

## Description: 

* 第1行: 输出1个整数，即FJ最少建立无线电通讯塔的数目 


## Sample Input: 
```
5
1 3
5 2
4 3
3 5
输入说明:
 Farmer John的农场中有5块草地：草地1和草地3相邻，草地5和草地2、草地
4和草地3，草地3和草地5也是如此。更形象一些，草地间的位置关系大体如下：
（或是其他类似的形状）
               4  2
               |  |
            1--3--5


```

## Sample Output: 
```
2
输出说明:
FJ可以选择在草地2和草地3，或是草地3和草地5上建通讯塔。
```
