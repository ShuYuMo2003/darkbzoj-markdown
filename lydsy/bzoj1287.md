# Cards交换卡片 - 题目 - Universal Online Judge


## Sample Input: 
```
3 4
2 1 0 0
0 2 0 0
1 0 3 1
```

## Sample Output: 
```
3
友情提示：小C有{1, 1, 2}, 第2个小朋友有{2, 2}, 第3个小朋友有{1, 3, 3, 3, 4}, 小C可以用一张1换第2个小朋友的一张2(第2个小朋友有2种不同的卡片了, 他会选择进行交换), 得到{1, 2, 2}, 再用1张2去换第3个小朋友的一张3, 得到{1, 2, 3}．他不可能得到4, 因此他最多的到3种不同的卡片．
```
