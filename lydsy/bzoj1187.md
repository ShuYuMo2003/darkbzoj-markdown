# [HNOI2007]神奇游乐园 - 题目 - Universal Online Judge

## Description: 

输出文件中仅一行为一个整数，表示最高的满意度之和。

### Input: 

大家测下这个数据 5 5 1 1 -100 3 3 1 1 -100 3 3 1 1 -100 3 3 1 1 -100 3 3 1 1 -100 3 3 结果是30?


## Sample Input: 
```
4 4
100 300 -400 400
-100 1000 1000 1000
-100 -100 -100 -100
-100 -100 -100 1000
```

## Sample Output: 
```
4000
```
