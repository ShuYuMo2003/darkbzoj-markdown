# [Ctsc2005]玩具的重量weight - 题目 - Universal Online Judge


## Sample Input: 
```
【样例输入1】
3 2
1 3 2 4 3 5
1 1 -1 1 2
1 1 1 2 3
【样例输入2】
3 1
1 5 2 5 1 3
2 1 1 1 2 3

```

## Sample Output: 
```
【样例输出1】
3 3 4 4 3 3
【样例输出2】
1 2 2 3 2 3
//样例1对应于题目描述中的例子。
//在样例2中，冰冰有三个玩具，重量的原始范围为1~5, 2~5, 1~3。只有一次称量，
天平的左边是玩具1和玩具2，右边是玩具3。左边比右边重1个单位。
根据此结果，可以判断三个玩具的重量范围为1~2，2~3和2~3
```
