# [Usaco2006 Open]Two-Headed Cows 双头牛 - 题目 - Universal Online Judge

## Description: 



### Input: 

第一行有两个整数，N和M。 下面M行，每一行有一对互不喜欢的头的情况。如4 A 3 B，表示4号牛的A头不喜欢3号牛的B头。

### Output: 

输出一个整数，最少的槽数。


## Sample Input: 
```
4 5
3 B 1 B
4 A 3 A
2 B 1 B
4 B 2 A
3 A 2 B



```

## Sample Output: 
```
2


```
