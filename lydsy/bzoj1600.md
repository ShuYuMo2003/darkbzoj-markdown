# [Usaco2008 Oct]建造栅栏 - 题目 - Universal Online Judge

## Description: 

*第一行：一个数n 

### Input: 

*第一行：合理的方案总数 


## Sample Input: 
```
6

```

## Sample Output: 
```
6
输出详解：
Farmer John能够切出所有的情况为: (1, 1, 1,3); (1, 1, 2, 2); (1, 1, 3, 1); (1, 2, 1, 2); 
(1, 2, 2, 1); (1, 3,1, 1);(2, 1, 1, 2); (2, 1, 2, 1); (2, 2, 1, 1); or (3, 1, 1, 1).
下面四种 
(1, 1, 1, 3), (1, 1, 3, 1), (1, 3, 1, 1), and (3,1, 1, 1) – 不能够组成一个四边形.
```
